# Alexa-skill
Alexa skill to record daily activity
This project focuses on creating a Alexa skill that can record daily activity whenever a user wants to. The DynamoDB saves the activity whenever 
the user calls the trigger and asks alexa to add the activity. Alexa can respond back with the activities performed by the user
whenever the user Triggers Alexa .This skill helps you monitor your daily activity and keep a check on it. Have used Lambda functions , DynamoDb and 
Alex skill set to do the project.
